The C# application Streamer is used to test the data transfer
on isochronous endpoints.

Load the firmware cystream.iic or cystream.hex in the device 
before running the application.


Streamer example support Visual studio 2008 and 2010. 

If you have Visual studio 2008 installed on your system , please use solution : \Streamer\Vs2008_sln\StreamerVS2008.sln to open the project. This solution will load the StreamerVS2008.csproj project file. 

If you have Visual studio 2010 installed on your system , please use solution : \Streamer\Vs2010_sln\StreamerVS2010.sln to open the project. This solution will load the StreamerVS2010.csproj project file.
