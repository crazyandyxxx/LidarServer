The C# Application Bulkloop is used to test the loop back of 
data transfer thruogh bulk endpoints.

load the firmware file, bulkloop.iic or bulkloop.hex in the folder Bulkloop.


BulkLoop example support Visual studio 2008 and 2010. 

If you have Visual studio 2008 installed on your system , please use solution : \BulkLoop\Vs2008_sln\BulkLoopVS2008.sln to open the project. This solution will load the BulkLoopVS2008.csproj project file. 

If you have Visual studio 2010 installed on your system , please use solution : \BulkLoop\Vs2010_sln\BulkLoopVS2010.sln to open the project. This solution will load the BulkLoopVS2010.csproj project file.
